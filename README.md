# UI Showcase

UI Showcase is a gallery of independent projects focused on translating designs into fully functional web interfaces. The emphasis is on:

- Accurate styling and layout

- Smooth, performant interactions

- Semantic, accessible HTML structure

*Live site:* **[https://ui-showcase-eight.vercel.app/]**

## Project Sources

Many projects are inspired by or adapted from:

- [Frontend Mentor](https://www.frontendmentor.io/)

- [Figma Community UI kits](https://www.figma.com/@prokit_design)

## Tech Stack

- Frameworks: React
- Styling: Tailwind CSS


## Structure

Each project lives in its own repository, making it easy to explore individually.

The showcase site provides links to these projects and presents them in a clean, navigable layout.
